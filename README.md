# LiraOS 🛸

**Assistant IA personnel pour Windows** — un compagnon vocal inspiré de LIRA,
l'IA de bord de *Star Citizen*. Tu branches l'IA de ton choix (OpenRouter, OpenAI,
Anthropic, Ollama local…) avec **ta** clé, et tu gardes la main sur tes données.

## 📥 Installation

1. Va dans **[Releases](../../releases/latest)** et télécharge le dernier `LiraOS-Setup-x.y.z.exe`
2. Lance l'installeur. Windows affiche « éditeur inconnu » (l'app n'est pas signée) :
   → **Informations complémentaires** puis **Exécuter quand même**
3. Au 1er lancement : **Réglages** → clé API + modèle

Les versions suivantes s'installent **automatiquement**.

## ✨ Aperçu

- 💬 Chat multi-fournisseurs en streaming (OpenRouter, Anthropic, OpenAI, Ollama)
- 🎭 Multi-personas, expressions, **Live2D** (personnage animé + lip-sync)
- 🎤 Voix : synthèse (Edge TTS) et reconnaissance vocale (Whisper local ou cloud)
- 🛠️ Outils : Spotify, Philips Hue, Home Assistant, recherche web, génération d'images
- ⚡ Commandes vocales & slash · 🔄 mises à jour automatiques

> 🔒 Certaines **fonctions avancées** (outils HTTP personnalisés, commandes-script)
> se débloquent avec une **clé de licence** (Réglages → Fonctions avancées).

## ⚙️ Configuration requise

- **Windows 10 / 11** (64 bits)
- Une **clé API** d'un fournisseur LLM (ex. [OpenRouter](https://openrouter.ai) — modèles gratuits dispo)
